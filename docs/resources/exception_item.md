# Resource elastic-siem-detection_exception_item

Exception item resource

## Example Usage

### Exception item YAML file (item.yml)

```yaml
item_id: hacker_item
list_id: hacker_list
name: Catch a hacker
description: This item excepts the specified user from alerting.
namespace_type: single
tags:
  - MyTag
type: simple
entries:
  - field: user.name
    operator: included
    type: match_any
    value:
      - hacker
expire_time: "2027-01-01T00:00:00.000Z"
```

### Terraform resource

```terraform
resource "elastic-siem-detection_exception_item" "exception_items" {
  for_each               = { for f in fileset(path.module, "*.yml") : f => file("${path.module}/${f}") }
  exception_item_content = jsonencode(yamldecode(each.value))

  # Helps syncing between objects
  depends_on = [elastic-siem-detection_exception_container.exception_lists]
}
```
