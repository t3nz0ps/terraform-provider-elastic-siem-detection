# Resource elastic-siem-detection_detection_rule

Detection rule resource

## Example Usage

### Rule YAML file (rule.yml)

Rule sample
```yaml
name: Rule to catch a hacker
rule_id: hacker_rule
description: This rule catches a hacker.
enabled: true
tags:
  - MyTag
false_positives:
  - A user might not be a hacker
license: My License
author:
  - John Doe
interval: 20m
from: now-10m
to: now
severity: low
risk_score: 21
max_signals: 100
index:
  - myindex-*
type: query
language: kuery
query: |-
  'user.name: "hacker"'
threat:
  - framework: MITRE ATT&CK
    tactic:
      id: TA001
      name: Initial Access
      reference: https://attack.mitre.org/tactics/TA0001/
    technique:
      - id: T1133
        name: External Remote Services
        reference: https://attack.mitre.org/techniques/T1133/
actions:
- id: <ID>
  params:
    message: |-
      Some message for concerned user

      {{{context.results_link}}}
  action_type_id: .teams
  frequency:
    summary: true
    notifyWhen: onActiveAlert
    throttle: null
  group: default
exceptions_list:
  - list_id: hacker
    id: <GENERATED_LIST_ID>
    namespace_type: single
    type: detection
```

Rule sample with type 'New Terms'
```yaml
name: Rule to catch a hacker (new_terms)

enabled: false
investigation_fields:
  field_names:
  - host.name
  - user.name
  - process.name

from: now-63m
to: now
severity: low
risk_score: 21
max_signals: 100
type: new_terms
language: kuery
index:
  - myindex-*
query: 'data_stream.namespace : "workstations" AND event.code: 1'
new_terms_fields:
  - process.name
history_window_start: now-14d
```

### Terraform resource

```terraform
resource "elastic-siem-detection_detection_rule" "kibana_security_rules" {
  for_each     = { for f in fileset(path.module, "*.yml") : f => file("${path.module}/${f}") }
  rule_content = jsonencode(yamldecode(each.value))

  # Helps syncing between objects
  depends_on = [elastic-siem-detection_exception_container.exception_lists]
}
```
